TooXTips — Logo Pack v1.0
=========================

Files included:

VARIANT A — App Icon
  toextips_icon.svg         → Dark background (app store, splash screen)
  toextips_icon_light.svg   → Light background

VARIANT B — Icon + Wordmark
  toextips_wordmark_dark.svg   → Dark background (headers, onboarding)
  toextips_wordmark_light.svg  → Light background

VARIANT C — Logotype (wordmark only)
  toextips_logotype_dark.svg   → Dark background (minimal usage)
  toextips_logotype_light.svg  → Light background

Colour Palette
--------------
  Violet (Primary)  #7C4DFF
  Cyan (Expenses)   #00C2D4
  Green (Practice)  #1D9E75
  Dark BG           #0D0D0F
  Light BG          #F2F2F7

Usage in Flutter
----------------
  Add flutter_svg to pubspec.yaml, then:
  SvgPicture.asset('assets/logos/toextips_icon.svg')

Format: SVG (vector, infinitely scalable)
